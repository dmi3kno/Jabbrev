Jabbrev

Marcus W. Beck, mbafs2012@gmail.com

This R package converts a journal name (or any text) to abbreviations using the ISSN List of Title Word Abbreviations (LTWA). Some journals insist that journal names in bibliographies are abbreviated. This is dumb so hereâ€™s a package that does it for you.

install.packages('devtools')
library(devtools)
install_github('fawda123/Jabbrev')

One journal at a time:

abbrev('Journal of Applied Mathematics')
## [1] "J. Appl. Math."

Or convert a whole .bib file:

bib_shr('inst/refs.bib')
